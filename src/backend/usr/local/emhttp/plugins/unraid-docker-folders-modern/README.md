**Docker Folders Modern**

Organize your Docker containers into folders with drag-and-drop, live resource stats, and real-time updates. A modern replacement for folder.view built with Vue 3.

- Drag-and-drop containers into custom folders
- Auto-group Docker Compose stacks
- Live CPU, memory, network, and I/O stats
- Real-time container status via WebSocket
- Grid and list view modes
- Import/export folder configuration
